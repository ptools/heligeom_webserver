Sources

The source code of the webserver is available on Github: https://github.com/ptools/heligeom_webserver

This webserver relies on a library called Ptools (1-2) whose source code is available here: https://github.com/ptools/ptools

The molecule viewer plugin used on this website is based on Mol* Viewer (3).

Citation

If you use the Heligeom webserver, please cite:

Santuz H, Laurent B, Robert CH, and Prévost C (2025) Heligeom: A web resource to generate, analyze, and visualize filament architectures based on pairwise association geometries of biological macromolecules. (submitted).
